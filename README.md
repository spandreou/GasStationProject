# Gas Station Shift Manager

Εφαρμογή διαχείρισης εβδομαδιαίων βαρδιών για πρατήριο καυσίμων με:
- React + Vite + Tailwind
- Drag and Drop με `dnd-kit`
- Firestore (με fallback σε localStorage αν λείπουν env vars)

## Εκκίνηση

```bash
npm install
npm run dev
```

## Firebase

1. Αντιγραφή `.env.example` σε `.env`
2. Συμπλήρωση Firebase τιμών.

Αναλυτικό schema: `FIREBASE_SCHEMA.md`.
